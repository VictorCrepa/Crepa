What is MUGEN?
M.U.G.E.N (also known as Mugen, MUGEN) is a freeware 2D fighting game engine designed by Elecbyte. Content is created by the community, and thousands of fighters, both original and from popular fiction, have been created. It is written in C and originally used the Allegro library. The latest versions of the engine now use the SDL library.
